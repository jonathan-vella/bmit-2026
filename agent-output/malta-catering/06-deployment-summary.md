### Validation Issues

No template or governance preflight issues blocked the deployment. The apply phase failed only when Azure attempted to create `cae-malta-catering-dev` in `swedencentral`.

Azure returned `ManagedEnvironmentCapacityHeavyUsageError` / `AKSCapacityHeavyUsage`:

> Creating a free tier cluster is unavailable at this time in region `swedencentral`. To create a new cluster, Azure recommends using an alternate region or creating a paid tier cluster.

Official Container Apps guidance confirms that `swedencentral` supports workload profiles, including paid Dedicated profiles (`D4`, `D8`, `D16`, `D32`, `E4`, `E8`, `E16`, `E32`) and `Flex`. The current Bicep implementation provisions the default serverless/consumption environment, so switching to a paid cluster requires a Step 5 Bicep change before Step 6 can continue.

Selected recovery path: move forward with a paid Container Apps workload profile in `swedencentral`.
